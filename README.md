# Videogame Publisher Simulator

Simulador de marketing y ventas para videojuegos en Steam desarrollado con Python, SimPy, Streamlit, Pandas, Matplotlib.

## Caracteristicas
- Modelado de alcance organico y pagado.
- Simulacion de Shock Events, como puede ser alcance por figuras grandes en internet como streamers, youtubers o critico.
- Algoritmo de conversion afectado por la retencion de usuarios.
- Calculo de ROI neto tras comisiones de Steam y reembolsos.

## Instalacion y Uso Local
1. Clonar el repositorio.
2. Instalar dependencias: pip install -r requirements.txt
3. Ejecutar: streamlit run app.py
